# Produit & Innovation - Geestock

## Positionnement actuel
- **Produit** : Sac magnétique pour bouteille
- **Cible** : Sportifs (running, trail, fitness, cyclisme)
- **USP** : Fixation magnétique = mains libres

## Améliorations produit court terme
1. **Vidéo de démonstration** : tourner ou commander une vidéo UGC montrant le produit en action
2. **Guide des tailles** : ajouter un tableau de compatibilité bouteilles
3. **FAQ enrichie** : répondre aux objections les plus courantes (aimant vs téléphone, poids max)

## Idées d'innovation (moyen terme)

### Produits complémentaires (cross-sell)
- Bouteille sport isotherme compatible (bundle naturel)
- Brassard sport pour téléphone (même cible)
- Serviette sport magnétique (même technologie)
- Ceinture de running avec porte-accessoires

### Extensions de gamme
- Version **XL** pour bouteilles 1.5L
- Version **Kids** avec couleurs et motifs enfants
- Version **Pro** avec compartiment clés/carte
- Éditions limitées saisonnières (couleurs été/hiver)

## Métriques produit à suivre
- **Taux de conversion** : objectif > 2% (visiteur -> achat)
- **Taux de retour** : objectif < 5%
- **NPS** (Net Promoter Score) : objectif > 50
- **Panier moyen** : objectif > 35 EUR (via bundles)
- **LTV** : objectif > 45 EUR (achat répété dans les 12 mois)

## Validation avant scaling
- [ ] 50 premières ventes avec taux de retour < 5%
- [ ] Au moins 10 avis clients positifs authentiques
- [ ] ROAS publicitaire > 3x sur 2 semaines consécutives
- [ ] Aucune réclamation majeure qualité produit
