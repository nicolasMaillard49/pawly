# Stratégie Publicitaire - Geestock

## Canaux recommandés (par priorité)

### 1. Meta Ads (Facebook/Instagram) - PRIORITAIRE
- **Budget test initial** : 20-30 EUR/jour pendant 7 jours
- **Objectif** : Conversions (Purchase)
- **Ciblage initial** :
  - France, 25-45 ans
  - Intérêts : fitness, running, trail, sport en plein air, cyclisme
  - Comportement : acheteurs en ligne
- **Structure de compte** :
  - Campagne 1 : Prospection (audiences froides)
  - Campagne 2 : Retargeting (visiteurs site, ajouts panier)

#### Créatifs recommandés
1. **Vidéo UGC** (15-30s) : démonstration du produit en situation sportive
2. **Before/After** : galère sans Geestock vs liberté avec
3. **Témoignage client** : face caméra, authentique
4. **Carrousel** : 4-5 slides montrant les situations d'usage

#### KPIs cibles
- CTR > 1,5%
- CPC < 1 EUR
- CPA < 10 EUR
- ROAS > 3x

### 2. TikTok Ads - SECONDAIRE
- **Budget test** : 20 EUR/jour
- **Format** : Spark Ads (boost de contenu organique)
- **Ciblage** : 18-35 ans, intérêts sport/fitness
- **Créatifs** : format natif TikTok, POV, transitions

### 3. Google Ads - TERTIAIRE
- **Shopping Ads** : une fois le flux produit configuré
- **Search Ads** : mots-clés "porte bouteille magnétique", "accessoire sport bouteille"
- **Budget** : 15 EUR/jour

## Tracking & Attribution
- **Pixel Meta** : à installer (conversion API côté serveur recommandé)
- **TikTok Pixel** : à installer
- **Google Analytics 4** : à configurer
- **UTM** : systématique sur tous les liens publicitaires

## Phase de test (4 semaines)
1. **Semaine 1-2** : Test de 5-8 créatifs différents, audiences larges
2. **Semaine 3** : Couper les créatifs CPA > 12 EUR, scaler les gagnants
3. **Semaine 4** : Lancer retargeting, tester lookalike audiences

## Budget mensuel recommandé
- **Phase test** : 600-900 EUR/mois
- **Phase scale** : 1500-3000 EUR/mois (si ROAS > 3x confirmé)
